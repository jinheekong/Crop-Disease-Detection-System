* 프로젝트 명 : 농작물 병해충 관리 AI 시스템
* 진행 기간 : 2025.01.13 ~ 2025.01.24
* 목적 : 객체 탐지 기술을 활용하여 농작물의 병해충 감염 유무를 탐지하고, 이를 휴대폰 어플리케이션 및 스프링쿨러 등을 이용하여 농장 관리 시스템을 구축해 장기간 외출시에도 보다 편리한 농장 관리를 가능하게 하기 위함.
